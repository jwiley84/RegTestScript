# RegTestScript

## Overview

This repository contains a rough batch file for regression testing the NODE ONLY samples for the Microsoft **Bot Builder V4 SDK** - [JS SDK](https://github.com//microsoft/botbuilder-js).

Samples for the Bot Builder SDK are available [here](https://github.com/Microsoft/BotBuilder-Samples).

## HOW TO

To use this file:

1.) Clone file to desktop.

2.) Open with preferred IDE. 

3.) Find all references to <MSBOT INFO HERE>

4.) Replace with information from own LUIS/QnA/appInsights/Dispatch

5.) Save file

6.) Double-click icon to run. 

Script pauses periodically, including just after the bot samples are started, to allow user to open emulator and test bot.
